Installing Python and IDLE

  1. Goto http://www.python.org/downloads.
  2. Click the yellow button that says 'Download Python 3.5.2.'
  3. Run the downloaded file once downloaded.
  4. Click 'Continue' on bottom right of installation window.
  5. Continue clicking 'Continue' on bottom right until it says 'Agree.'
  6. Click 'Agree' to continue the installation.
  7. Click 'Install' on bottom right of installation window.
  8. Enter password if prompted.

    The installation will begin and make take a few minutes. Once finished, you will see:

    If you see this, the installation was successful and you can continue. If you see something else, try going back to Step 3 and reinstalling Python.
